Car Wash Management System is a Flutter-based application designed to simplify and automate car wash service operations.
It allows customers to book services online and enables admins to manage users, services, and bookings efficiently.

Features:
User
Register & Login
View Services
Book Car Wash
View Booking History
Check Bill Amount
Submit Reviews & Support Tickets

Admin
Manage Services
Manage Users
Manage Bookings
View Reviews
Dashboard Control

Technology Used
Flutter
Dart
Database (MySQL / Backend)
